# Rough Heston Model Parameters Calibration

Python implementation for pricing European vanilla options and calibrating the rough Heston model to market data (S&P 500). Based on the fractional Riccati equation and the fractional Adams method described in * El Euch, O., & Rosenbaum, M. (2016). *The characteristic function of rough Heston models*. arXiv preprint [arXiv:1609.02108](https://arxiv.org/abs/1609.02108).
